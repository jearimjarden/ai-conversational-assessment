# Design System

## Purpose

This document defines the visual design system for the AI Conversational Assessment Agent.

The goal is to provide a clean, modern, and professional user experience that reflects the Lightrees brand while maintaining consistency across all screens.

---

# Design Principles

The interface should be:

- Clean
- Professional
- Friendly
- Minimalist
- AI-first
- Mobile responsive
- Accessible

The AI should feel like a professional consultant, not a chatbot.

---

# Brand Personality

LIA should feel like:

- Professional
- Helpful
- Trustworthy
- Intelligent
- Calm
- Efficient

Avoid making the interface look playful or overly futuristic.

# UI Consistency Rules

- Use the same spacing throughout the application.
- Keep button styles consistent.
- Do not mix different icon styles.
- Use cards for grouped information.
- Display progress throughout the assessment.
- Keep interactions predictable.
